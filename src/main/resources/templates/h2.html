H2 데이터베이스 설치

개발이나 테스트 용도로 가볍고 편리한 DB, 웹 화면 제공

H2 homepage에서 다운로드 및 설치

h2 데이터베이스 버전은 스프링 부트 버전에 맞춘다.

권한 주기 : `chmod 755 h2.sh`

실행 : `./h2.sh`

데이터베이스 파일 생성 방법 : `jdbc:h2:~/test`(최초한번) -> `~/test.mv.db` 파일 생성 확인 -> 이후부터는 'jdbc:h2:tcp://locahost/~/test`로 접속

h2 데이터베이스 버전은 스프링 부트 버전에 맞춘다. 스프링 부트 2.x를 사용하면 1.4.200 버전, 스프링 부트 3.x를 사용하면 2.1.214 버전

테이블 생성

        drop table if exists member CASCADE;
        create table member
        (
         id bigint generated by default as identity,
         name varchar(255),
         primary key (id)
        );

더미 데이터 삽입

        insert into member(name) values('spring');
        insert into member(name) values('java');

데이터 조회

        select * from member;

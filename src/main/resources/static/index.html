스프링 부트가 제공하는 Welcome Page

스프링 부트는 스프링 생태계 자체 감싸서 편리하게 사용할 수 있도록 도와주는데, 자바 엔터프라이즈 웹 어플리케이션과 관련된 전반의 생태계를 다 제공하기 때문에 어마어마하게 크다.
